Flask REST API Fundamentals

Project Overview

This project demonstrates the fundamentals of REST API development using the Flask framework in Python. It provides simple API endpoints to retrieve and add student data using JSON format.

Features

Built with Python Flask
GET endpoint to retrieve student information
POST endpoint to add new student data
JSON request and response handling
RESTful API concepts implementation
API Endpoints

Home Endpoint

GET /

Returns a welcome message.

Get Students

GET /students

Returns a list of student records.

Add Student

POST /students

Adds a new student record and returns a success message.

Technologies Used

Flask
JSON
